FAQ - Perguntas Frequentes

Última atualização: 22 de junho de 2025

O SaaS Maps utiliza o OpenStreetMap (Nominatim + Overpass) como motor de busca principal, permitindo encontrar estabelecimentos comerciais em qualquer cidade do Brasil sem custos. Quando disponível, também consulta o Google Places API para enriquecer os resultados com dados adicionais. Basta selecionar o estado, a cidade e a categoria desejada para iniciar a busca.
Oferecemos três planos: Basic (grátis, até 100 leads/mês), Pro (R$49/mês, até 1.000 leads com IA e Google Places) e Enterprise (R$199/mês, leads ilimitados com suporte prioritário). Todos os planos incluem busca via OpenStreetMap e exportação CSV. Consulte nossa página de preços para detalhes completos.
Sim, cada plano tem um limite mensal: 100 leads no Basic, 1.000 no Pro e ilimitado no Enterprise. Os limites são resetados no primeiro dia de cada mês. Você pode acompanhar seu uso atual no painel do dashboard. Se atingir o limite, poderá fazer upgrade do plano para continuar salvando leads.
O OpenStreetMap (OSM) é um mapa colaborativo e gratuito, similar ao Google Maps, mas mantido por uma comunidade global de voluntários. Usamos os serviços Nominatim (para geolocalização) e Overpass (para consulta de estabelecimentos) para buscar dados atualizados sem custos de API. Isso nos permite oferecer o plano Basic gratuito.
Não é obrigatório. O sistema funciona 100% com OpenStreetMap. No entanto, se você configurar uma chave do Google Places API no arquivo config.js, os resultados serão enriquecidos com dados adicionais como avaliações, horários e fotos. A chave pode ser obtida gratuitamente no Google Cloud Console.
Após realizar uma busca, os resultados aparecem em uma tabela. Clique no botão "Salvar" na coluna à direita para adicionar o estabelecimento à sua lista de leads salvos. Você também pode gerenciar os status (Novo, Contatado, Em negociação, Cliente, Descartado) e adicionar notas nos leads salvos.
Após realizar uma busca, clique no botão "Exportar CSV" localizado abaixo da tabela de resultados. O arquivo será baixado automaticamente com todos os dados dos estabelecimentos encontrados, incluindo nome, telefone, website, endereço e status. A exportação inclui todos os resultados filtrados no momento.
Usuários dos planos Pro e Enterprise têm suporte prioritário via chat direto no dashboard. Usuários do plano Basic podem enviar dúvidas através dos canais disponíveis no repositório oficial do projeto. Para questões urgentes, recomendamos abrir uma issue no GitHub do projeto.
Os dados do OpenStreetMap são atualizados continuamente pela comunidade. Quando você realiza uma busca, os dados são obtidos em tempo real da fonte. O cache de cidades do IBGE é armazenado localmente para agilizar as buscas, mas pode ser atualizado manualmente limpando o cache do navegador.
